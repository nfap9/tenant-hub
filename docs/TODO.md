## 三、轻量级短板（5项）🟢

| 项目               | 说明                                                                    | 修复成本 |
| ------------------ | ----------------------------------------------------------------------- | -------- |
| Token 自动刷新     | 后端 `POST /auth/refresh` 已实现，前端 `apiClient` 未接入自动刷新机制   | 低       |
| 退出登录（服务端） | `POST /auth/logout` 已实现，`AppSessionContext.signOut()` 未调用        | 低       |
| 组织头像/信息修改  | `PUT /organizations/:id` 已实现，前端 `OrganizationPage` 编辑功能已添加 | ✅       |
| 个人资料修改       | `PUT /auth/me` 已实现，`AccountPage` 的修改信息表单仍是空壳             | 低       |
| 手机号直邀成员     | `POST /organizations/:id/members` 已实现，`MembersPage` 未添加入口      | 中       |
