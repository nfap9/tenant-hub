## 三、前端使用演示桩（2处）🔴

**后端 API 已存在，但前端未连接。**

| 位置                | 问题                                                            | 修复方案                                           |
| ------------------- | --------------------------------------------------------------- | -------------------------------------------------- |
| **MembersPage.tsx** | `handleRoleChange` 显示 `message.success('角色已更新（演示）')` | 调用 `PUT /organizations/:id/members/:userId/role` |
| **MembersPage.tsx** | `handleRemove` 显示 `message.success('成员已移除（演示）')`     | 调用 `DELETE /organizations/:id/members/:userId`   |

## 四、后端 API 缺失（7处）🔴

| FR         | 缺失端点                          | 说明                                              |
| ---------- | --------------------------------- | ------------------------------------------------- |
| **FR-004** | `PUT /auth/me`                    | 个人资料修改无后端 API（用户名/头像等无法保存）   |
| **FR-005** | `POST /auth/logout`               | 退出登录仅清本地缓存，无服务端标记                |
| **FR-005** | `GET /auth/me/sessions`           | 会话管理完全缺失                                  |
| **FR-007** | `POST /organizations/:id/members` | 通过手机号直接邀请成员（现有仅邀请码方式）        |
| **FR-009** | `GET /organizations/:id`          | 组织详情查询                                      |
| **FR-702** | `GET /bills/:id`                  | 账单详情端点（前端 MonthlyDetailPage 调用会 404） |
| **FR-705** | `PUT /bills/:id/write-off`        | 坏账核销                                          |
